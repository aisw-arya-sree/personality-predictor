Personality Predictor App

The Personality Predictor App is a machine learning project that classifies a person as an Introvert or Extrovert based on their social habits and behavioral patterns. It uses a Random Forest Classifier trained on structured data and is deployed as an interactive web app using Streamlit.

Key Features

Supervised learning classification

Data preprocessing and categorical encoding

Model training and evaluation

Real-time prediction through a simple UI

Technologies

Python, Pandas, NumPy, Scikit-learn, Streamlit

Output

Predicts personality type: Introvert / Extrovert
